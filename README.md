## おすすめアプリ 3 選

### Focus Pet - Habbit
> タスク完了とフォーカス時間でペットを育てる、ゲーム感覚のタスク管理アプリ

- タスク完了・フォーカスセッションで XP を獲得してペットが進化
- 報酬システムで習慣化をサポート
- ゲーム感覚で TODO 管理を楽しく続けられる

[![App Store](https://img.shields.io/badge/App_Store-Focus_Pet-0D96F6?logo=apple&logoColor=white)](https://apps.apple.com/us/app/focus-pet-habbit/id6759491280)

---

### Asset Forecast
> 資産・家計の状況を"天気マーク"で直感的に把握できるスマート資産管理アプリ

- 晴れ・曇り・雨で毎月の資産コンディションを一目で確認
- 50 種類以上の指標・グラフを自由に配置できる分析画面
- データは全て端末内にのみ保存（アカウント不要・完全オフライン）

[![App Store](https://img.shields.io/badge/App_Store-Asset_Forecast-0D96F6?logo=apple&logoColor=white)](https://apps.apple.com/jp/app/asset-forecast/id6753914294)

---

### SeedPocket
> 家庭菜園のハーブ・野菜とレシピを一つのタイムラインで管理する"暮らしのノート"

- 栽培スケジュール・収穫ログをレシピ候補と連動させて可視化
- 畑ではアプリで記録、自宅の PC からは Web で管理
- モバイルアプリ（iOS）と Web の両方で利用可能

[![App Store](https://img.shields.io/badge/App_Store-SeedPocket-0D96F6?logo=apple&logoColor=white)](https://apps.apple.com/us/app/seed-pocket/id6756086853)
[![Web App](https://img.shields.io/badge/Web_App-seedpocket.net-22d3ee?logo=googlechrome&logoColor=white)](https://seedpocket.net/)

## このリポジトリについて

AquaPeak 公式サイト（[aquapeakstudio.com](https://aquapeakstudio.com/)）のソースコードです。

**技術スタック**
- Next.js 16 (App Router) + React 19
- Tailwind CSS v4
- TypeScript

**ローカル起動**

```bash
npm install
npm run dev   # http://localhost:3000
```
